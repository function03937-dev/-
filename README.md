# FoodBridge Rescue Project

## How to Run This Program:

1. **Open your terminal or command prompt** and navigate to this folder.

2. **Install the required libraries** by running:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   python app.py
   ```
   *(Note: If `python` doesn't work, try `python3 app.py`)*

4. **Open your web browser** and go to:
   http://127.0.0.1:8000/

The first time you run the app, it will automatically create a fresh SQLite database (`foodbridge2.db`) for you!
